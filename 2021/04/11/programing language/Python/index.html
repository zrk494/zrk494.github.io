1.1 输入输出

  • print()，用于表示输出

1.2 变量和常量

1.2.1 变量

  • 变量无需像其他语言一样提前声明

  • 可以同时给多个变量赋值

  • 一种特殊的数据类型—分数

1.3 运算相关

  • 算数运算

  • lambda（自定义一种简单的运算）
