4. 索引文件

4.1 基本概述

  • 索引 一种数据结构，是有序文件
  • 索引 索引文件之中的信息包括两个：
    • 存储的数据文件索引域的值
    • 记录所在的地址

4.2 分类方式（通过各种类型的索引理解如何建立索引，不同分类中的索引可能部分重合）

4.2.1 分类1

  • 稠密
  • 稀疏

4.2.2 分类2

  • 单极
  • 多级

4.2.3 分类3

  • 主索引

    • 索引域是关系的主键
    • 数据文件和索引文件都是按主键的值进行排序
    • 是一种稀疏索引

  • 聚集索引

    • 索引项不是文件的键属性
    • 数据文件和索引文件都按照索引域进行排序
    • 是一种稀疏索引

  • 辅助索引

    ​

  • 建立在数据文件的非排序域之上

  • 索引域值对应一个数据记录

  • 是一种稠密索引
