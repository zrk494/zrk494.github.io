2. 磁盘文件

2.1 数据项

  • 表示元组之中属性的值

2.2 文件记录

  • 数据项的集合/文件记录对应一个元组
  • 分类
    1. 定长记录
    2. 非定长记录
  • 存储方法
    1. 一个文件存储于多个文件块
    2. 一个文件存储在一个文件块（定义在后面）

2.3 文件块

  • 文件记录的集合
  • 文件块的集合构成文件，文件对应一个关系

2.4 有序文件

  • 有序文件指的是记录按某个域的值的大小进行排序，这个域为排序域
  • 查找操作
  • 插入操作
  • 删除操作
  • 修改操作

2.5 无序文件

3. Hash文件

4. 索引文件

4.1 基本概述

  • 索引 一种数据结构，是有序文件
  • 索引 索引文件之中的信息包括两个：
    • 存储的数据文件索引域的值
    • 记录所在的地址

4.2 分类方式（通过各种类型的索引理解如何建立索引，不同分类中的索引可能部分重合）

4.2.1 分类1

  • 稠密
  • 稀疏

4.2.2 分类2

  • 单极
  • 多级

4.2.3 分类3

  • 主索引

    • 索引域是关系的主键
    • 数据文件和索引文件都是按主键的值进行排序
    • 是一种稀疏索引

  • 聚集索引

    • 索引项不是文件的键属性
    • 数据文件和索引文件都按照索引域进行排序
    • 是一种稀疏索引

  • 辅助索引

    ​

  • 建立在数据文件的非排序域之上

  • 索引域值对应一个数据记录

  • 是一种稠密索引
